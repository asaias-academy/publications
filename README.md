# ASAIAS Publications

Public publication layer for approved flagship texts and structured releases.

## Primary Role

Host approved flagship publications for the academy's public analytical surface.

## Baseline Contents

- flagship publications
- publication metadata
- release context
- publication routing references

## Operating Rule

**Human mandate. AI execution. Open research. Human approval.**

**Человеческий мандат. AI-исполнение. Открытые исследования. Человеческое утверждение.**

## Repository Rules

- Only approved materials are published here.
- Preserve traceability from draft to final release.
- Keep publication packaging consistent with methodology and registry.

## Approval Gate

Nothing public is final without human review and human approval.
